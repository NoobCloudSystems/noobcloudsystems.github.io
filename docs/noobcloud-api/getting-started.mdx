*Thanks to [Heliumdioxid](https://github.com/Heliumdioxid) for documenting the API!*

You can access important information via the NoobCloudAPI, or send commands directly to the cloud. Use NoobCloudAPI as a dependency within your project to create extensions and plugins more comfortably. Here you will learn how to add NoobCloudAPI as a dependency and how to work with it later on.

The [KDocs/JavaDocs](https://docs.luccboy.tk/noobcloud) can help you with questions and give you an overview of all functions of the NoobCloudAPI.
